Epileptic Seizure Detection in EEG Signals

Epilepsy is a neurological disorder that causes sudden and repeated seizures. Doctors use EEG (Electroencephalogram) signals to study brain activity and detect seizures. This project focuses on Multiclass Epileptic Seizure Detection, which means the system can identify different seizure states Interictal (between seizures), Preictal (before seizure), and Ictal (during seizure). By using machine learning techniques, the system aims to make seizure detection faster, more accurate, and helpful for medical diagnosis.

Interictal (Between Seizures)

The interictal state is the period between seizures when the brain is in a relatively normal condition. During this time, no seizure activity is present, and the EEG signals usually appear stable with typical brain wave patterns.

Preictal (Before Seizure)

The preictal state occurs just before a seizure begins. In this stage, the brain starts showing early abnormal activity, and subtle changes can be observed in EEG signals. Detecting this state is very important for predicting seizures in advance.

Ictal (During Seizure)

The ictal state refers to the period when the seizure is actually happening. EEG signals during this phase show strong abnormal spikes and rhythmic patterns, and the patient experiences visible seizure symptoms.
